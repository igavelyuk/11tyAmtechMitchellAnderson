#  11ty blog Amtech

### Base install
`git clone git@github.com:igavelyuk/11tyAmtechMitchellAnderson.git`

### How to use
#### Optimise
`npx @11ty/eleventy`
#### Serve
`npx @11ty/eleventy --serve`


#### Packages
npm install --global del-cli
